- Use 8px grid system

**Layout & Grid:**

### Detailed Component Specifications

**1. Hero Section:**
- Full-width with subtle gradient or very light mesh background
- Large centered or left-aligned headline
- Sub-headline directly below with good breathing room
- Two prominent CTAs (primary button in cyan, secondary in outline style)
- Professional headshot on the right side (or subtle abstract Web3 visual)
- Very high visual weight on name and role

**2. Trust Bar:**
- Thin horizontal bar with exchange names or logos
- Subtle divider lines
- Centered or justified layout
- Small text size with good contrast

**3. Power Stats Section:**
- Use large, bold numbers (minimum 48–64px)
- Each stat in its own clean card or in a clean 3–6 column grid
- Very short supporting label below each number
- Strong visual separation but elegant
- This section must feel like the strongest credibility moment on the page

**4. Cards (Track Record, Projects, etc.):**
- Clean rounded corners (12–16px radius)
- Subtle border (#1F2937)
- Soft inner shadow or no shadow (keep it minimal)
- Hover effect: very subtle lift + border color change to accent
- Consistent icon + title + description structure

**5. Animations & Interactions:**
- Use subtle Framer Motion style transitions
- Stats numbers can have a gentle count-up animation on scroll
- Buttons: scale 1.02 + shadow on hover
- Cards: very light lift on hover
- No aggressive animations — keep everything elegant and calm

**6. Overall Visual Hierarchy:**
- Hero → Trust Bar → Power Stats must be the first three things the user sees
- Heavy use of whitespace between major sections
- Clear visual breaks between sections using subtle dividers or spacing
- All important credibility elements (stats, backing, exchange experience) must be extremely prominent
